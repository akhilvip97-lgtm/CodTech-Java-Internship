# Java File Handling Utility

## Objective
This project demonstrates Java file handling operations such as:

- File Creation
- File Writing
- File Reading
- File Modification
- File Deletion

## Technologies Used
- Java
- VS Code

## Features
- Menu-driven program
- Exception handling
- User-friendly console interface

## How to Run

1. Compile:
javac src/*.java

2. Run:
java -cp src Main
